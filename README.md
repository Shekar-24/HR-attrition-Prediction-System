# Employee Attrition Prediction using Machine Learning

## Overview
Employee Attrition Prediction is a Machine Learning project designed to identify employees who are likely to leave an organization. The project analyzes HR-related factors such as job satisfaction, salary, overtime, work-life balance, performance, and experience to help companies improve employee retention using data-driven insights.

This project demonstrates a complete Machine Learning workflow including data preprocessing, exploratory data analysis, feature engineering, model training, hyperparameter tuning, and model evaluation.

---

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering and encoding
- Handling imbalanced data using SMOTE
- Multiple Machine Learning models implementation
- Hyperparameter tuning and cross-validation
- Performance comparison of models
- Feature importance analysis
- Employee attrition risk prediction
